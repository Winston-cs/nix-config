# Index

## Academic.nix
> notes, emails and more

This file has the **academic** tag. Enabling it installs:

- Obsidian.md
- Zotero
- Thunderbird
- LibreOffice
- Zoom.us

## Dev.nix
> clis, coding tools, and more

This file has the **dev** tag. Enabling it installs:

- All clis and coding languages
- Zed (mainly used for LEAN 4)
- All cli editors
- Godot
- Kitty

## Games.nix
> video games

This file has the **games** tag. Enabling it installs:

- Prism Launcher and Lunar Client
- HSR
- Steam
- Vesktop

## Media.nix
> media players and art stuff

This file has the **media** tag. Enabling it installs:

- MPV
- Krita
- Halftone
- gThumb
- Pinta
